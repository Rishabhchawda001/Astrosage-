
=== Chapter 1 ===

oṃ śaṃ no mitraḥ śaṃ varuṇaḥ / śaṃ no bhavatvaryamā / śaṃ na indro bṛhaspatiḥ / śaṃ no viṣṇururukramaḥ / namo brahmaṇe / namaste vāyo / tvameva pratyakṣaṃ brahmāsi / tvāmevapratyakṣaṃ brahma vadiṣyāmi / ṛtaṃ vadiṣyāmi / satyaṃ vadiṣyāmi / tanmāmavatu / tadvaktāramavatu / avatu mām / avatu vaktāram // oṃ śāntiḥ śāntiḥ śāntiḥ // iti śikṣādhyāye prathamo 'nuvākaḥ //
oṃ śīkṣāṃ vyākhyāsyāmaḥ / varṇaḥ svaraḥ / mātrā balam / sāma santānaḥ / ityuktaḥ śīkṣādhyāyaḥ // iti dvitīyo 'nuvādakaḥ //
saha nau yaśaḥ / saha nau brahmavarcasam / athātaḥ saṃhitāyā upaniṣidaṃ vyākhyāsyāmaḥ / pañcasvadhikaraṇeṣu / adhilokamadhijyautiṣamadhividyamadhiprajamadhyātmam / tā mahāsaṃhitāyā ityācakṣate / athādhilokam / pṛthivī pūrvarūpam / dyauruttararūpam / ākāśaḥ sandhiḥ //
