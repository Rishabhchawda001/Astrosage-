# Chunk Quality Skill

## Purpose
Score chunks across 9 quality dimensions.

## Core Module
`core/chunk_quality/quality.py`

## Dimensions
Completeness, Evidence, Confidence, Structure,
Metadata, Hierarchy, Citation, Language, Graph Connectivity

## Grades
A (≥0.9), B (≥0.8), C (≥0.6), D (≥0.4), F (<0.4)
